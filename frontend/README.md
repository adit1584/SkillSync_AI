## Overview

The SkillSync AI frontend is a modern, premium React single-page application (SPA) that provides:

- **Resume upload & parsing** — drag-and-drop PDF/DOCX upload with live feedback
- **AI-driven skill gap analysis** — visual radar charts and match scoring
- **Interactive quiz engine** — adaptive multi-choice skill verification quiz
- **Career dashboard** — simulation overview with readiness scores
- **Learning roadmap** — phase-based timeline with milestones
- **Course recommendations** — curated courses linked to skill gaps
- **AI mock interview** — real-time chat-based technical interview with scorecard
- **ATS resume optimizer** — AI-generated bullet points to improve resume ranking
- **PDF report export** — print-ready full-results report

## Pages

### `LandingPage.jsx`
The hero marketing page. Features animated headline, feature cards, and a prominent CTA leading to the upload page. Uses GSAP for scroll-triggered entrance animations.

### `UploadPage.jsx`
Resume upload flow with a drag-and-drop zone (React Dropzone) and a **target role selector** covering 30+ engineering, data, design, and product roles. Submits the resume to the backend for AI parsing.

### `QuizPage.jsx`
Adaptive skill verification quiz. Renders multiple-choice questions generated by the AI based on the user's parsed resume. Tracks scores and navigates to results on completion.

### `ResultsPage.jsx`
The main dashboard after analysis. Contains **6 tabbed sections**:
| Tab | Component |
|---|---|
| Overview | `CareerDashboard` |
| Skill Simulation | `CareerDashboard` (sim view) |
| Learning Roadmap | `RoadmapTimeline` |
| Courses | `CourseRecommendations` |
| AI Mock Interview | `InterviewSimulator` |
| Resume Optimizer | `ATSResumeOptimizer` |

Also includes a **Download PDF** button that fetches all tab data concurrently and triggers a clean `window.print()` report.

## Components

### `Navbar.jsx`
Fixed top navigation with the SkillSync AI logo, dark/light theme toggle, and navigation links. Has a `.print-hide` class so it is excluded from PDF exports.

### `ResumeUploader.jsx`
Drag-and-drop file upload with file type validation, size check, and upload progress feedback.

### `CareerDashboard.jsx`
Displays career simulation results — match scores, skill radar chart, gap breakdown, and interview readiness indicators.

### `SkillRadarChart.jsx`
Recharts-based radar chart visualizing skill coverage across multiple domains.

### `RoadmapTimeline.jsx`
Phase-by-phase learning timeline with milestones, estimated durations, and resource links. Rendered as a vertical timeline with animated phase cards.

### `CourseRecommendations.jsx`
Curated course cards with platform badges (Coursera, Udemy, YouTube, etc.), difficulty levels, and direct links.

### `InterviewSimulator.jsx`
Real-time terminal-style mock interview chat. Sends user responses to the AI interviewer, tracks conversation history, and displays a final scorecard with overall score, technical depth, communication rating, strengths, and improvement areas.

### `QuizEngine.jsx`
Renders adaptive quiz questions with animated option selection and score tracking.

### `ATSResumeOptimizer.jsx`
Generates ATS-optimized resume bullet points based on skill gaps and the target role.

## Getting Started

### Prerequisites
- **Node.js** v18+
- **npm** v9+
- Backend server running on `http://localhost:5000` (see `/backend/README.md`)

### Installation

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at **http://localhost:3000**.

> The Vite dev server proxies all `/api/*` requests to `http://localhost:5000`, so no CORS configuration is needed during development.

---

## Environment Variables

Create a `.env` file in the `frontend/` directory:

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

> `.env` is listed in `.gitignore` and should **never** be committed to version control.

## Design System

The entire visual system is defined in [`src/index.css`](./src/index.css) using **CSS custom properties (variables)**.

### Color Palette

```css
/* Light Mode */
--primary:       #C2714F   /* Terracotta */
--accent:        #8B4513   /* Warm brown */
--bg-primary:    #FAF6F0   /* Warm sand */
--bg-secondary:  #F5EFE6   /* Light sand */
--text-primary:  #2D1810   /* Deep brown */

/* Dark Mode (data-theme="dark") */
--bg-primary:    #121212   /* Material light-black */
--bg-secondary:  #1E1E1E
--text-primary:  #F0EAE0
```

### Typography
- **Headings**: Space Grotesk (Google Fonts)
- **Body**: Inter (Google Fonts)
- **Code/Terminal**: JetBrains Mono

### Theme Toggle
The theme is managed by `ThemeContext` (in `src/context/`). The `<Navbar>` renders a toggle switch that sets `data-theme="dark"` on the `<html>` element.

### Print / PDF Export
A `@media print` block in `index.css` hides all `.print-hide` elements (Navbar, tab bar, buttons) and forces all result sections to display as a clean vertical document.

## API Integration

All backend calls are centralised in [`src/lib/api.js`](./src/lib/api.js) using an Axios instance.

### Key Endpoints

| Function | Method | Endpoint | Description |
|---|---|---|---|
| `uploadResume` | POST | `/api/upload` | Upload resume PDF/DOCX |
| `analyzeSkills` | POST | `/api/analyze` | AI skill gap analysis |
| `fetchQuiz` | GET | `/api/quiz/:id` | Fetch quiz questions |
| `submitQuiz` | POST | `/api/quiz/submit` | Submit quiz answers |
| `fetchSimulation` | GET | `/api/simulate/:id` | Career simulation data |
| `fetchRoadmap` | GET | `/api/roadmap/:id` | Learning roadmap |
| `fetchCourses` | GET | `/api/courses/:id` | Course recommendations |
| `startInterview` | POST | `/api/interview/start` | Begin AI mock interview |
| `chatInterview` | POST | `/api/interview/chat` | Send interview response |
| `fetchOptimizer` | GET | `/api/optimizer/:id` | Resume optimizer suggestions |
